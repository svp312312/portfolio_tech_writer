# Группировка профилей

## Как это работает?

Яндекс объединяет ваши профили в разных социальных сетях, чтобы показывать пользователям полную и актуальную информацию о вас. Чтобы профили были сгруппированы, нужно подтвердить, что они действительно принадлежат одному человеку. Яндекс считает профили связанными, если между ними есть двусторонняя, подтверждённая ссылка.  *Подтверждённой* считается ссылка, которая требует авторизации в другой сети с входом по паролю.

## Самый простой способ: сервис «Поиск по людям»

Если у вас есть аккаунт на Яндексе, самый удобный способ — воспользоваться сервисом [social.yandex.ru](https://social.yandex.ru/).  Проставлять взаимные ссылки в самих профилях в этом случае не обязательно.   В противном случае - при отсутствии аккаунта на Яндексе, вы можете войти как пользователь соцсети. Не забудьте разрешить поисковой системе использовать информацию о связях между профилями.
